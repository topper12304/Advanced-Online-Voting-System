# 🚀 Quick Start Guide - Advanced Online Voting System

## ⚡ Fastest Way to Run

```
1. Open VS Code
2. Right-click "index.html"
3. Click "Open with Live Server"
4. Done! 🎉
```

- Feature cards showcasing system capabilities
- "Start Voting" button to begin

## 🔧 Troubleshooting

### Live Server Not Working?
1. Install "Live Server" extension in VS Code
2. Search in Extensions (Ctrl+Shift+X)
3. Install by Ritwick Dey

### Page Not Loading?
- Check browser console (F12)
- Make sure all 3 files are in same folder:
  - index.html
  - styles.css
  - script.js

### Already Voted Error?
- Refresh the page to reset
- Or use a different Voter ID
