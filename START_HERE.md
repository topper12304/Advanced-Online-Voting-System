## рҹҶҳ TROUBLESHOOTING

### Problem: Live Server not in right-click menu
**Solution:** Install Live Server extension from VS Code marketplace

### Problem: Page shows but looks broken
**Solution:** Make sure `styles.css` and `script.js` are in same folder as `index.html`

### Problem: "Already voted" error
**Solution:** Refresh the page or use a different Voter ID

### Problem: Nothing happens when clicking buttons
**Solution:** Check browser console (F12) for errors
