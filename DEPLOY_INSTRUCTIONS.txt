═══════════════════════════════════════════════════════════════════════
                    📤 STEP 1: PUSH TO GITHUB
═══════════════════════════════════════════════════════════════════════

1. Go to https://github.com/new
2. Create a new repository (name it "voting-system")
3. Don't check any boxes (no README, no .gitignore)
4. Click "Create repository"

5. In VS Code terminal, run these commands:

   git init
   git add .
   git commit -m "Complete voting system"
   git remote add origin https://github.com/YOUR-USERNAME/voting-system.git
   git branch -M main
   git push -u origin main


═══════════════════════════════════════════════════════════════════════
                    ⚙️ STEP 2: ENABLE GITHUB PAGES
═══════════════════════════════════════════════════════════════════════

1. Go to your repository on GitHub
2. Click "Settings" (top menu)
3. Click "Pages" (left sidebar)
4. Under "Source":
   - Select "Deploy from a branch"
   - Branch: "main"
   - Folder: "/ (root)"
5. Click "Save"


═══════════════════════════════════════════════════════════════════════
                    ⏳ STEP 3: WAIT FOR DEPLOYMENT
═══════════════════════════════════════════════════════════════════════

Wait 2-3 minutes for GitHub to deploy your site.

You'll see a message like:
"Your site is live at https://YOUR-USERNAME.github.io/voting-system/"


═══════════════════════════════════════════════════════════════════════
                    🎉 YOUR LIVE URL WILL BE:
═══════════════════════════════════════════════════════════════════════

    https://YOUR-USERNAME.github.io/voting-system/

    Replace YOUR-USERNAME with your actual GitHub username!

═══════════════════════════════════════════════════════════════════════
                    🔄 UPDATING YOUR SITE
═══════════════════════════════════════════════════════════════════════

To update your live site after making changes:

   git add .
   git commit -m "Updated voting system"
   git push

GitHub will automatically redeploy (takes 2-3 minutes).

═══════════════════════════════════════════════════════════════════════
                    🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Problem: 404 Error
Solution: Wait 5 minutes, ensure repository is public

Problem: Changes not showing
Solution: Wait 2-3 minutes, clear browser cache (Ctrl+Shift+R)

Problem: Site not deploying
Solution: Check Settings → Pages → Actions tab for errors
