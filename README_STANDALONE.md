# Advanced Online Voting System - Standalone Version

## 🚀 Quick Start

1. Open the project folder in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"
4. The website will open in your browser automatically!

## 🔧 No Installation Required!

This is a standalone version that runs directly in your browser.
No Node.js, npm, or build tools needed!

## 📝 Notes

## 🎨 Customization

You can easily customize:
- Colors in `styles.css` (see `:root` variables)
- Candidates in `script.js` (CANDIDATES array)
- Voter data in `script.js` (MOCK_VOTERS object)
